# 🏃‍♀️ Endless Runner

Um pequeno jogo 2D de **corrida infinita** desenvolvido na **Unity**, onde o jogador precisa correr o máximo possível enquanto desvia de obstáculos e coleta itens pelo caminho.  
Foi um projeto criado para treinar lógica, organização de scripts e a construção de mecânicas simples, mas bem conectadas.

---

## ✨ O que foi feito

- Sistema de **movimento do jogador** (corrida e pulo)
- **Reposição automática do chão (Ground Repeat)**
- **Parallax** no fundo para dar sensação de movimento
- **Criação e movimentação de obstáculos**
- **Coletáveis** simples com pontuação
- Configuração básica de **áudios e feedbacks**

---

## 💡 O que ainda pode melhorar

- Adicionar uma **tela de início** para o jogo  
- Implementar um **sistema de pause**  
- Criar **novos tipos de coletáveis** com efeitos diferentes  
- Incluir **variação de tipos de chão** para deixar o cenário mais dinâmico  
- Adaptar os controles para **mobile**

---

## 🎮 Sobre o projeto

Esse projeto foi desenvolvido com base nos vídeos da [**Academia Unity Desenvolvimento de Jogos - GameDev**](https://www.youtube.com/watch?v=aS5ZoP3dctA&list=PLciEA-M4VWn0ofKL0ddLeauOhMmGmEPQ8).  
Por se tratar de um material mais antigo, realizei **ajustes na programação** para que o código ficasse compatível com versões mais recentes da Unity e seguisse práticas mais atuais.

Foi uma ótima oportunidade para reforçar conceitos de **movimento, repetição de cenário, parallax e organização de scripts** dentro de um projeto completo e funcional.
